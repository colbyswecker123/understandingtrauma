# Sleek Style Upgrade

This version adds a sticky transparent gradient header with a glass/blur effect.

Main style upgrades:

- Header stays at the top while the page scrolls
- Header is slightly transparent with a gradient and blur
- Logo and hamburger are more compact
- Menu drawer feels more premium
- Cards and buttons are softer and less heavy
- Footer is lighter and less boxed-in
- Cache-busted to v10 so mobile browsers reload the new styles

Test with:

https://understandingtrauma.org/?v=10

The header uses backdrop-filter with a fallback for browsers that do not support blur.
