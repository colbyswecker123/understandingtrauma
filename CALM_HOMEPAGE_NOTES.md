# Calmer Homepage Redesign

This version intentionally simplifies the home page.

Homepage now focuses on:

1. The note
2. Show another message
3. Submit a custom message
4. A small calm strip for music/reset
5. One quiet "Need something else today?" section

Store, Support, Faith, Resources, and Destress still exist, but they are no longer oversized homepage buttons.

Footer links were simplified to:

Resources
Faith
Support

Admin remains accessible through the hamburger menu and direct /admin/ URL.
